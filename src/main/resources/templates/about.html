About This Project

A modern Java Spring Boot application showcasing Azure Container Apps deployment with full CI/CD automation

Technology Stack

Java 17

Modern Java runtime with Spring Boot framework for robust enterprise applications

Spring Boot

Production-ready framework with embedded Tomcat, Thymeleaf templating, and actuator endpoints

Azure Container Apps

Serverless container hosting with auto-scaling from 0-2 replicas based on demand

Docker

Containerized deployment using Eclipse Temurin JDK images with multi-stage builds

GitHub Actions

Automated CI/CD pipeline with Docker build, ACR push, and Container Apps deployment

Terraform

Infrastructure as Code for Azure resources with state management and version control

Architecture Overview

Azure Container Registry (ACR)

Private Docker image repository with Basic SKU, integrated with GitHub Actions for automated image builds and storage.

Azure Container Apps Environment

Managed Kubernetes environment with Log Analytics Workspace for monitoring, metrics, and application insights.

Azure Storage Account

Terraform state backend storage ensuring consistent infrastructure state management across deployments.

Features & Capabilities

Blog Platform

Full-featured blog with posts, tags, search functionality, and responsive design

Health Monitoring

Spring Boot Actuator endpoints for health checks, metrics, and application monitoring

REST API

RESTful endpoints for programmatic access to blog posts and application information

Responsive Design

Mobile-first CSS design with modern UI/UX patterns and accessibility features

Security

Azure Managed Identity integration and secure container deployment practices

Auto-scaling

Automatic horizontal scaling based on CPU and memory usage with cost optimization

Development Journey

Initial Setup

Started with basic Spring Boot application and Azure Web App deployment attempt

Migration to Container Apps

Encountered quota limitations and migrated to Azure Container Apps architecture

ACR Integration

Implemented Azure Container Registry for private image storage and management

CI/CD Pipeline

Automated deployment pipeline with GitHub Actions and infrastructure provisioning

Blog Enhancement

Transformed simple REST API into full-featured blog website with modern styling
